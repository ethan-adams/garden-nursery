# Garden Nursery Game

A cozy systems game about running a regional plant nursery, learning what local gardeners want, and creating new varieties with care.

## First Prototype

Open `index.html` in a browser. The first version is a small web game with:

- A weekly nursery loop.
- Region-specific customer demand.
- Plant inventory with care traits.
- Simple selling, restocking, and hybrid experiments.
- A trend/taste panel that nudges the player toward market reading.

## Design Pillars

- Gardening should feel observant, patient, and joyful.
- Plant knowledge should matter, but the game should stay approachable.
- The market should feel regional: climate, taste, season, and neighborhood habits shape demand.
- Hybridizing should feel like discovery instead of pure spreadsheet optimization.

## Near-Term Ideas

- Add a propagation bench and greenhouse layout.
- Introduce customer archetypes: apartment balcony gardeners, native-plant enthusiasts, chefs, collectors, landscapers.
- Track local weather and seasonal events.
- Make hybrids inherit traits more visibly.
- Add a journal for discovered varieties and market notes.
